# 阿片类药物急危重症知识库

本知识库来源于《阿片类药物在急危重症中的应用专家共识》（中华急诊医学杂志 2024年）及 **118 个真实临床病例**，涵盖 ACS、主动脉夹层、重症胰腺炎、颅脑创伤、烧伤、机械通气、癌性疼痛、术后疼痛、阿片相关并发症等场景。

> 为什么必须先查库再作答：训练数据中存在过时结论（例如"哌替啶首选胰腺炎"已被推翻），本知识库收录最新共识，直接用训练数据回答可能给出错误的临床建议。

数据文件位于本 skill 的 `data/` 目录，欢迎通过 GitHub PR 贡献新病例（见 CONTRIBUTING.md）。

---

## 第一步：查真实病例库

从 118 个病例中检索最相关案例，获取推荐方案、循证依据、风险提示。

```bash
python "<此SKILL.md所在目录>/scripts/search_cases.py" "<查询词>"
```

查询词示例：
- `"急性主动脉夹层 高血压"`
- `"颅脑创伤 机械通气 芬太尼"`
- `"重症胰腺炎 镇痛"`
- `"癌性疼痛 吗啡 滴定"`
- `"阿片过量 纳洛酮 救治"`

返回字段：病例ID、诊断、推荐方案、循证依据、风险提示、治疗结果

---

## 第二步：查专家共识原文

从共识OCR全文中检索相关段落，获取原始推荐级别和证据等级。

```bash
python "<此SKILL.md所在目录>/scripts/search_guideline.py" "<查询词>"
```

查询词示例：
- `"氢吗啡酮 胰腺炎"`
- `"吗啡 Oddi括约肌 禁忌"`
- `"芬太尼 颅内压 机械通气"`
- `"哌替啶 不推荐"`

---

## 第三步：组织回答

两个工具都查完后，用以下结构作答（**必须**标注病例来源）：

```
**推荐方案**
[具体药物、剂量、途径]

**循证依据**
[来自共识的推荐条款 + 证据等级]

**风险提示**
[禁忌证、特殊人群注意事项]

**参考病例**
【CASE-XXX】[病例简述]
```

---

## 降级处理

- 如果病例库未找到匹配 → 仅凭共识原文作答，注明"本次未找到匹配病例，以下依据来自专家共识"
- 如果共识也未找到 → 说明知识库暂无该场景的记录，建议查阅原始文献，不要凭训练数据给出具体用药建议
- 两个工具均无结果时，切勿自行编造循证依据或推荐方案
