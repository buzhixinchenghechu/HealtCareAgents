# 贡献指南 / Contributing Guide

欢迎为阿片类药物临床知识库贡献病例数据！

---

## 如何添加新病例

编辑 `data/sample_cases.json`，在数组末尾追加一个新的病例对象：

```json
{
  "id": "CASE-XXX",
  "diagnosis": "诊断名称",
  "category": "癌性疼痛 | 术后疼痛 | 急危重症镇痛 | 阿片相关并发症 | 其他疼痛",
  "pain_type": "急性 | 慢性 | 癌性",
  "comorbidities": "合并症描述",
  "recommended_plan": "推荐药物、剂量、途径、频次",
  "evidence": "循证依据来源（指南名称、发表年份）",
  "risk_notes": "风险提示、禁忌证、特殊人群注意事项",
  "outcome": "治疗结果描述"
}
```

### 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| `id` | ✅ | 格式：`CASE-A01`（急危重症）、`CASE-C01`（癌痛）、`CASE-P01`（术后）、`OP-001`（其他） |
| `diagnosis` | ✅ | 主要诊断，尽量具体 |
| `category` | ✅ | 从上方枚举值中选择 |
| `recommended_plan` | ✅ | 包含药物名、剂量、途径、频次 |
| `evidence` | ✅ | 引用来源，如"中华急诊医学杂志2024年专家共识" |
| `risk_notes` | ✅ | 禁忌证或注意事项 |
| `pain_type` | 可选 | 疼痛分类 |
| `comorbidities` | 可选 | 合并症 |
| `outcome` | 可选 | 治疗结果 |

---

## 数据质量要求

- 病例来源须为**真实临床场景**或**权威指南推荐案例**
- `recommended_plan` 须包含具体剂量（如 `吗啡 5mg IV q4h`）
- `evidence` 须注明文献来源，不接受"经验用药"
- 禁止虚构病例数据

## 提交方式

1. Fork 本仓库
2. 编辑 `skill/medical-opioid/data/sample_cases.json`
3. 提交 Pull Request，标题格式：`data: 添加[诊断名称]病例 (CASE-XXX)`

## 知识库当前覆盖场景

| 类别 | 病例数 |
|------|--------|
| 急危重症镇痛（ACS、夹层、胰腺炎、TBI、烧伤等） | ~15 |

| 术后疼痛（PCIA/PCA、多模式镇痛） | ~20 |
| 阿片相关并发症（过量、纳洛酮救治、成瘾） | ~20 |
| 特殊人群（老年、肝肾功能不全、妊娠哺乳） | ~15 |
| 神经病理性疼痛、慢性疼痛 | ~18 |
| **合计** | **118** |
