# 成瘾治疗用药知识库

本知识库包含 **5000+ 真实临床病例**的成瘾治疗用药方案，涵盖阿片类成瘾、酒精成瘾、冰毒成瘾等场景。

## 数据概览

- **成瘾类型**: opioid（阿片类）、alcohol（酒精）、meth（冰毒）
- **治疗药物**: buprenorphine（丁丙诺啡）、methadone（美沙酮）、naltrexone（纳曲酮）等
- **数据字段**: 问题、推荐方案、患者年龄、诊断、药物

数据文件位于 `data/addiction_cases.csv`

---

## 第一步：查询病例库

根据患者年龄和成瘾类型检索匹配的用药方案：

```bash
python "<此SKILL.md所在目录>/scripts/search_cases.py" "<查询词>"
```

查询词示例：
- `"27岁 opioid成瘾"`
- `"40岁 alcohol成瘾"`
- `"meth成瘾 naltrexone"`
- `"buprenorphine 剂量"`

---

## 第二步：组织回答

查询完成后，用以下结构作答：

```
**推荐方案**
• 药物：[药物名称]
• 剂量：[具体剂量]mg
• 频次：[QD/BID/TID]
• 治疗分组：[A/B]

**禁忌症**
[根据药物类型列出禁忌]

**注意事项**
• 老年/肝肾功能不全者减量
• 严禁与酒精、苯二氮䓬类联用
• 需在专业医师监护下使用

**监测指标**
• 生命体征、呼吸频率
• 戒断症状评分
• 药物依从性与不良反应
```

---

## 药物速查

| 药物 | 适应症 | 主要禁忌 |
|------|--------|----------|
| buprenorphine | opioid/alcohol成瘾 | 过敏者禁用 |
| methadone | opioid成瘾 | 严重呼吸抑制、肝功能衰竭 |
| naltrexone | opioid/meth/alcohol成瘾 | 过敏者禁用 |

---

## 降级处理

- 如果病例库未找到精确匹配 → 返回相似年龄段或相同成瘾类型的案例作为参考
- 如果完全无匹配 → 说明知识库暂无该场景记录，建议咨询专业医师
- 切勿自行编造用药方案或剂量
